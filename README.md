# 📻 MyRadio

お気に入りのインターネットラジオをボタン一つで聴けるパーソナルラジオプレイヤー。
サーバー不要・ブラウザだけで動作する静的 Web アプリ。

---

## 特徴

- **ページ内再生** — 別タブに飛ばず、そのまま再生
- **カテゴリーフィルター** — ジャンル別に絞り込み可能
- **2 種類のボタン**
  - 🟡 アンバー：直接ストリーム再生
  - 🔵 ブルー：公式サイトを別タブで開く（直接再生不可の局）
- **モバイル対応** — スマホ・タブレットでも使用可能
- **サーバー不要** — HTML ファイル 1 枚で完結

---

## ラジオ局の追加・編集・削除

`index.html` 内の `STATIONS` 配列を編集するだけ。

```js
const STATIONS = [
  // 🟡 直接再生できる局（アンバーボタン）
  {
    name: "局の名前",
    url: "https://ストリームURL.mp3",
    category: "カテゴリー名",
    desc: "説明文（省略可）"
  },

  // 🔵 直接再生できない局（ブルーボタン・別タブで開く）
  {
    name: "局の名前",
    website: "https://公式サイトURL",
    category: "カテゴリー名",
    desc: "説明文（省略可）"
  },
];
```

### ルール

- `url` があれば → アンバーボタン（ページ内再生）
- `website` があれば → ブルーボタン（別タブ）
- `category` は自由に決めて OK。同じ名前を使うと自動でグループ化される
